# mevn-item
Simple CRUD with NodeJS, ExpressJS, MongoDB, and VueJS+Vuex

## Clone repository
```sh
$ git clone https://github.com/ardisaurus/mevn-item
```

## Install and run the back-end
```sh
$ cd mevn-item/server
$ npm install
$ npm run dev
```

## Install and run the front-end
```sh
$ cd mevn-item/client
$ npm install
$ npm run serve
```
